This plugin provides advanced settings to display recent posts
== Description ==
Advanced Last Posts plugin is a light plugin, which allows for increased customization for displaying recent posts from multiple categories or from custom post types.

Features:

1.Plugin

- Add New - you can use add new post with featured image.
- Category - you can view and add new Category. 
- Options - you can choose where from you can use posts (From plugin or default Posts Wordpress).
- Help - lists of shortcodes.

2.Widget
- Title - Name of the section.
- Number of Posts - display number of last posts.
- Show posts thumbnail - featured image will display when this checkbox is checked.
- Thumbnail size - It will only work when 'Show posts thumbnail' is checked.
- Posts excerpt in words - Number of words to display. It works only if Posts excerpt is checked. 
- Read more text - You can display "read more" text. It works only if Posts excerpt is checked.

** .Works with placing a shortcode.



== Installation ==

1. Upload `Advanced Last Posts` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set some default settings in Advanced Last Posts -> Options in your admin and enable the use of the plugin
4. Use `[inventolp]` shortcode in your Post/Page or place `<?php echo do_shortcode( '[inventolp]' ) ?>` in your templates.
